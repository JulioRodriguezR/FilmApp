# FilmApp

Store cinephile information with the movie app built with [Ionic](https://ionicframework.com/).

## Key Features
- Main view based on: _News_, _Billboard_ y _Popular_.
- Obtaining detail such as description and cast of actors.
- Search on database.
- Manage of favorites.
